# react-native-pop-ups

Pop ups that you required to notify your user

## Installation

```sh
npm install react-native-pop-ups
```

## Usage

```js
import PopUps from "react-native-pop-ups";

// ...

const result = await PopUps.multiply(3, 7);
```
